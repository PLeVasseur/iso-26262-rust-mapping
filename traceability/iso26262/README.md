# ISO 26262 Anchor Registry

This directory holds tracked, non-verbatim ISO anchor metadata used by the
Sphinx traceability pipeline.

- Canonical anchor catalog: `traceability/iso26262/index/anchor-registry.jsonc`
- PDF-set metadata: `traceability/iso26262/index/source-pdfset.jsonc`
- Relevance policy: `traceability/iso26262/index/relevant-pdf-policy.jsonc`
- Part indexes: `traceability/iso26262/parts/2018-ed2/*/index.jsonc`

No copyrighted ISO clause text is stored here.
